The Consortium for the Advancement of Scientific Software (CASS) has its first virtual Birds-of-a-Feather (BoF) days on July 11-12. The event is hosted by the Partnering for Scientific Software Ecosystem Stewardship, [PESO](https://pesoproject.org/), project. The event website can be found at [https://cass.community/news/2026-02-10-cass-bof-days.html](https://cass.community/bofs).

S4PST projects contributed and sponsored, along with other CASS members, the following sessions:

- Kokkos Quo Vadis
- MPICH: A High-Performance Open Source MPI Library for Leadership-class HPC Systems
