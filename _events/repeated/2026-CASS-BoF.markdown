The Consortium for the Advancement of Scientific Software (CASS) has its second virtual Birds-of-a-Feather (BoF) days on February 10-11. The event is hosted by the Partnering for Scientific Software Ecosystem Stewardship, [PESO](https://pesoproject.org/), project with the participation of other CASS members. Registration and program information can be found at the [CASS event website](https://cass.community/news/2026-02-10-cass-bof-days.html)

S4PST projects contributed and sponsored the following sessions:

- Kokkos Quo Vadis
- MPICH: A High-Performance Open Source MPI Library for Leadership-class HPC Systems
